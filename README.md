# My Weather Simple
Weather Forecast App using REST API with Retrofit

Features:
- MVVM Architecture
- Retrofit2
- Uses REST API to fetch weather forecast and geocoding
- GPS location for forecast
- Coroutines
- Data and Viewbinding
